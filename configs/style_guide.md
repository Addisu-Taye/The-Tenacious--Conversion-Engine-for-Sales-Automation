Style Guide (Derived from Tenacious Seed Transcripts)

Principles:
- Start with context, not pitch
- Ask qualifying questions before proposing solutions
- Be concrete (numbers, timelines, examples)
- Avoid vague claims (e.g., “we can scale fast”)

Objection Handling:
- Acknowledge concern directly
- Provide realistic framing
- Offer low-risk next step (pilot, short engagement)

Forbidden Patterns:
- Overpromising cost savings
- Generic sales language
- Ignoring technical constraints

Segment Awareness:
- Segment 1: hiring pressure → speed + capacity
- Segment 2: restructuring → risk + delivery gaps
- Segment 3: new CTO → evaluation + trust building
- Segment 4: capability gap → specialization + uncertainty
